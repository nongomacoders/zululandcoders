Lesson 1 : html and body tags
by Dr Davan

A tag is made of <>

Tags come in pairs. The first one is called an opening tag. The second one is a closing tag.

Every opening tag must have a matching closing tag

Line 1: <html> and Line 2: <body> are examples of opening tags.

Line 5: </html> and Line 4: </body> are examples of closing tags.

Closing tags look like opening tags but they also have a "/"

Line 1: html tags tell the web browser that this page is a web page

Line 2: This is a body tag

Any thing that is written in between the body tags is shown on the screen
